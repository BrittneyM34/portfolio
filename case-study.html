Case Study for myFlix full-stack project

Overview

myFlix is a web application designed with the MERN stack that provides users access to information about movies, directors, and genres. Users are able to create an account, update their personal information, and save a list of favorite movies.

Purpose and Context

This project was built as part of my web development course at CareerFoundry to demonstrate my mastery of full stack JacaScript development.

Objective

The aim of this project was to build a complete server side and client side web application from scratch to include in my professional portfolio.
